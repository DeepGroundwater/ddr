# References

This folder contains reference datasets from external sources used by DDR for training, evaluation, and benchmarking. These include gage information files that map USGS streamflow gages to the river network, as well as scripts for downloading and formatting lateral inflow predictions.

## Supported Geospatial Datasets

- NOAA-OWP Hydrofabric v2.2
- MERIT Hydro Vectorized Flowlines
